VIGNASHREE SPARKLE WORD PLAY – VERSION 7 – INSTALLABLE TABLET APP

MAIN CHANGES
- App renamed to Vignashree Sparkle Word Play.
- Full-screen welcome page with Vignashree's photograph.
- All spelling-out of letters has been removed. Words are only pronounced.
- Exercise reading now has a Stop Reading button.
- Telugu audio uses direct online Telugu audio first and device Telugu speech as fallback.
- Read Your Chapter now has sentence-by-sentence navigation and Auto Teacher controls.
- Lessons 1 and 2 include verified step-by-step English reading with Telugu explanations.
- Lessons 3–7 contain safe placeholders until verified sentence scripts are supplied.
- Exercise Practice has seven common sections for every Reader chapter:
  Comprehension, Let's Think, Let's Listen, Let's Speak, Let's Create, Let's Write and Let's Apply.
- A separate Work Book tab has been added with all available workbook topics.
- Games are selectable chapter-wise and include Meaning Match, Missing Letter and Listen and Choose.

OPENING
1. Extract the ZIP file.
2. Open index.html in Chrome or Edge.
3. Keep internet available for Telugu online audio.
4. Click Explore My Learning on the welcome screen.

NOTE ABOUT TELUGU VOICE
Browser and operating-system Telugu voices vary. This version first attempts a direct online Telugu audio source.
If the network or browser blocks that source, it falls back to an installed Telugu system voice.


NEW IN VERSION 7 – INSTALLABLE TABLET APP
- Learn Words now includes “Sentences with This Word”.
- Each word displays three example sentences.
- The selected word is highlighted inside every sentence.
- Each sentence has English reading and Telugu-meaning audio.
- “Read All” reads each English sentence followed by its Telugu meaning.
- Common words have natural, child-friendly examples.
- Other glossary words receive simple reading-practice sentences automatically.
